# Lingeswara S - Personal Portfolio

Welcome to the repository for my personal developer portfolio! This project is a modern, highly responsive, and fully interactive web portfolio designed to showcase my skills as a Full Stack Developer specializing in Python, Django, and modern web applications.

## 🚀 Features

- **Modern UI/UX:** A sleek design featuring a glassmorphism aesthetic, smooth CSS animations, and floating orbs.
- **Light/Dark Mode:** Built-in theme toggling that completely transforms the color palette.
- **Interactive Sections:**
  - **Hero:** Typing animations and glowing, responsive elements.
  - **About & Skills:** Detailed statistics, biography, and categorized skill progress bars.
  - **Projects:** Interactive cards linking to live demos and GitHub repositories.
  - **Achievements:** A scalable certification grid featuring a "View More" toggle and a custom image lightbox.
- **Integrated Chatbot:** A custom JavaScript-powered portfolio assistant to help visitors navigate the site.
- **Fully Responsive:** Carefully crafted media queries ensure the layout looks perfect on everything from large desktop monitors to small mobile screens.
- **Contact Form:** Ready-to-use form layout for direct communication.

## 🛠️ Technology Stack

This portfolio is built entirely without heavy frontend frameworks to ensure maximum performance and complete control over the design system:
- **HTML5:** Semantic structuring.
- **CSS3:** Heavy use of CSS Variables (Custom Properties), CSS Grid, Flexbox, and complex keyframe animations (`variables.css`, `layout.css`, `hero.css`, etc.).
- **Vanilla JavaScript:** DOM manipulation, theme toggling, lightbox logic, dynamic skill loading, and chatbot interactions (`main.js`, `chatbot.js`, etc.).
- **Font Awesome:** For scalable vector icons.
